# Bodymind Tracker

A modern web app for tracking daily physical and mental activities.

## Features
- Today and tomorrow task views
- Step/set completion tracking
- Persistent local progress
- Weekly and monthly analytics
- Categories for body and mind activities
- Mobile-friendly UI

## Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Recharts
- Docker + Traefik deploy

## Local run
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
```
